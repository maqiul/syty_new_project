## 📝 使用说明

1. **启动前端：**
   ```bash
   cd I:\syty\syty-web
   npm run dev
   ```
   访问 `http://localhost:3000`

2. **测试流程：**
   - 登录 → 选择店铺 → 查看库存列表
   - 点击"入库"按钮 → 填写表单 → 提交
   - 点击"调整"按钮 → 填写表单 → 提交
   - 进入财务报表 → 查看统计卡片
   - 查看订单列表 → 测试筛选功能

3. **如果报错：**
   - 跨域问题：检查后端 CORS 配置，确保允许 `localhost:3000`
   - 404 错误：接口路径可能不匹配，查看浏览器 Network 面板
   - 数据格式不对：检查后端返回结构与前端类型定义是否一致
